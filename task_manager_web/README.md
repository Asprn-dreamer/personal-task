# FlowTask 个人任务管理网站

这是一个纯静态的个人任务管理网站，可直接运行在浏览器中，数据保存在 `localStorage`。

## 已实现功能

- 任务新增、编辑、删除
- 任务标题、描述、截止日期、优先级、状态、标签、置顶
- 按状态、优先级、标签、搜索关键词筛选
- 按智能排序、优先级、截止时间、创建时间、手动排序
- 今日任务、即将到期、已完成视图
- 子任务、拖拽排序、清理已完成
- 统计卡片、日历视图、完成记录
- 深色模式

## 运行方式

直接双击 `index.html` 即可打开。

也可以在本目录运行：

```powershell
python -m http.server 8080
```

然后访问 `http://localhost:8080`。
